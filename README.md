# 追影：贸易全球化中的碳责任追踪

一个围绕全球碳责任展开的交互式数据文档。

在线访问：  
https://re1nyyy.github.io/Global-Carbon-Accountability-Tracker/

本项目试图回答一个问题：当我们讨论“碳责任”时，是否只看一个国家或企业排放了多少碳，还是应该进一步追问这些排放由什么经济活动、政策选择、消费需求和企业生产结构共同造成？

页面通过四个互相独立但叙事连续的交互模块，帮助读者从政策、经济、贸易和企业四个角度追踪碳责任的形成与转移。

## 项目内容

### 1. 政策与排放拐点

将全球 CO2 排放趋势与重大国际气候政策放在同一时间轴上，观察《京都议定书》《巴黎协定》等政策节点出现后，全球排放是否真的发生变化。

读者可以拖动年份轴或点击政策节点，查看政策前后三年全球排放均值的变化。

### 2. GDP 与碳排放

关注经济增长是否必然带来更高排放。

项目使用 GDP、CO2 排放和国家发展类型数据，展示不同国家在不同阶段是否出现“经济增长但排放增速下降”的脱钩现象。读者可以通过气泡图、时间滑杆和国家筛选，比较不同经济体的增长模式。

### 3. 贸易与碳责任

关注全球贸易中的碳责任转移。

传统的领土排放只统计“烟囱在哪里”，但消费足迹会进一步追问“这些排放服务了谁的消费”。通过对比领土排放、消费足迹和净进口碳，本项目展示一些国家看似完成了本土减排，但其消费活动仍可能通过进口商品引发大量境外排放。

地图使用人均指标着色，总量信息通过悬停提示补充。读者可以切换年份和责任视角，观察哪些国家是碳责任转嫁者，哪些国家承担了更多出口生产中的碳成本。

### 4. 企业与排放责任

将视角从国家扩展到企业主体。

通过 Carbon Majors 数据，项目展示排放责任在少数大型化石燃料企业和高排放企业中的集中程度，帮助读者理解国家与贸易之外的产业责任结构。

## 主要特性

- 单页交互式数据文档
- 四个独立可视化模块
- 支持点击文本触发图表变化
- 支持地图视角切换、年份滑杆和悬停说明
- 使用 Vega-Lite、D3 和原生 JavaScript 实现
- 适配 GitHub Pages 静态部署
- 支持本地静态服务器运行

## 数据来源

本项目主要使用以下公开数据源：

- Our World in Data / Global Carbon Budget
  - 国家年度 CO2 排放
  - 消费端 CO2 排放
  - GDP 与人口数据
- Climate Policy Database
  - 国际气候政策与政策节点
- UNFCCC 官方资料
  - 关键国际气候协议说明
- Carbon Majors Database
  - 主要化石燃料企业和高排放企业数据
- Vega Datasets
  - `world-110m` 世界地图 TopoJSON 底图

说明：不同数据源的统计口径并不完全一致。本项目更强调在一致口径下进行相对比较，用于揭示碳责任结构，而不是给出绝对精确的责任判决。

## 本地运行

由于页面需要加载本地 CSV 数据文件，建议使用静态服务器运行，而不是直接双击打开 `index.html`。

在项目根目录运行：

```bash
python -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000
```

## 项目说明

本项目为“大数据可视化与可视分析”课程设计作品。它不是一个完整的政策评估模型，而是一个交互式数据叙事原型，目标是帮助读者从多个角度理解全球碳责任的复杂性。
